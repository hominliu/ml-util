## Development

### Source code
```commandline
git clone https://github.com/hominliu/ml-utils.git
```

### Installation
Install in editable mode with `[dev]` extra.
```commandline
pip install -e .[dev]
```

### Commit changes
We use **pre-commit** ensure coding style consistency.
Install pre-commit in local environment with
```commandline
pre-commit install
```
